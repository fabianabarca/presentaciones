Pruebas de REPL con PyScript

¿Qué es PyScript?

PyScript es un lenguaje de programación que permite la ejecución de código Python en el navegador web.

Este es un REPL.

Aquí podés escribir y modificar código.

2+2

Y correrlo con Shift+Enter.

Podés hacer operaciones, funciones, imprimir en pantalla y usar librerías.

def printText(myText): print(myText) printText("edit this")

Librerías

Podés usar librerías de ciencia de datos como numpy, scipy, matplotlib y pandas.

import numpy as np import scipy as sci import pandas as pd import matplotlib.pyplot as plt

Plots

Podés generar plots de matplotlib.

import matplotlib.pyplot as plt import numpy as np t = np.arange(0.0, 2.0, 0.01) s = 1 + np.sin(2*np.pi*t) plt.plot(t, s) plt.show()

Y ver el resultado aquí.

Tablas

También podés imprimir dataframes de pandas.

import pandas as pd df = pd.DataFrame({'num_legs': [2, 4, 8, 0], 'num_wings': [2, 0, 0, 0], 'num_specimen_seen': [10, 2, 1, 8]}, index=['falcon', 'dog', 'spider', 'fish']) print(df)
