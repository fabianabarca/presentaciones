Pruebas de REPL con PyScript

¿Qué es PyScript?

PyScript es un lenguaje de programación que permite la ejecución de código Python en el navegador web.

incluir link de pyscript

¿Qué es un REPL?

Un Read-Eval-Print Loop (REPL) es un entorno interactivo que permite escribir, ejecutar y ver resultados de código de forma inmediata.
