Pruebas de REPL con PyScript

¿Qué es PyScript?

PyScript es un lenguaje de programación que permite la ejecución de código Python en el navegador web.

import micropip import js from pyodide.ffi import create_proxy async def micropip_install(): await micropip.install(["numpy", "pandas", "matplotlib", "scipy"]) async def main(): await micropip_install() js.setTimeout(create_proxy(main), 0)
