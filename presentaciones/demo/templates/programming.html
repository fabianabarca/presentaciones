Pruebas de Ace.js con PyScript

¿Qué es PyScript?

PyScript es un lenguaje de programación que permite la ejecución de código Python en el navegador web a partir del uso de tecnologías web modernas como Pyodide y WASM (Web Assembly). Está diseñado para ser accesible a usuarios de todos los niveles, con muchísimas aplicaciones por su relación con Python.

Más información sobre PyScript aquí.

¿Qué es Ace.js?

Ace es un editor de código que se puede integrar en páginas web mediante JavaScript. Cuenta con características como resaltado de sintaxis y señalación de errores para más de 120 lenguajes de programación. Se utiliza en Cloud9 IDE y es el sucesor del proyecto Skywriter (Bespin) de Mozilla.

Más información sobre Ace.js aquí.

¿Qué es un REPL?

Un REPL (Read-Eval-Print Loop) es un entorno interactivo que permite escribir código, evaluarlo y ver los resultados inmediatamente en el mismo entorno.

Con Ace.js y Pyscript podemos simular un REPL. Lo que hacemos es esperar a que se presione el botón de correr para ejecutar el código en el editor como si fuera un script de PyScript.
