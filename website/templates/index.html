Una plataforma web de presentaciones para la enseñanza de probabilidad y estadística en ingeniería

Este sitio es resultado del Proyecto de Docencia PD-IE-472-2022 de la Escuela de Ingeniería Eléctrica, inscrito en la Vicerrectoría de Docencia de la Universidad de Costa Rica, denominado:
Estrategias docentes para sesiones virtuales interactivas con el desarrollo de un nuevo sistema web: una experiencia en el curso Modelos Probabilísticos de Señales y Sistemas
El proyecto desarrolla un sistema web de presentaciones para la enseñanza de probabilidad y estadística en ingeniería, con el objetivo de mejorar la experiencia de aprendizaje de los estudiantes del curso en la modalidad virtual, explotando la versatilidad del diseño web.

Para ingresar a presentaciones de ejemplo con las posibilidades del sistema, puede visitar la sección Demostraciones.

Para conocer más sobre sus objetivos, metodología y resultados, puede visitar la sección Sobre el proyecto.
